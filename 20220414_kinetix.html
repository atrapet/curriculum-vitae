Mes Parcelles - WEB

Refonte du header et du menu

Le header contenant la barre de menu devenait vieillissant et nécessitait une refonte. Ces deux élements étants le point d'entrée pour tout le reste de l'application il était important de réussir la refonte.

La barre de menu était auparavant gérée en html et js ainsi que php pour la génération, elle a été entièrement refondue en html css avec un json détaillant l'arborescence à la place de tableaux php. Flexbox à été utilisé pour correctement aligner l'ensemble. La hiérarchie a également été revue pour plus de clareté.

Le reste du header a été passé en responsive et le css de l'ensemble de ses élements a été repris pour moderniser l'aspect.

Mes Parcelles est un portail fréquenté par des milliers d'utilisateurs quotidiennement.
